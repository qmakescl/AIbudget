# 2026 AI 예산 대시보드 구현 계획

HTML/JS + Chart.js 기반 정적 대시보드로, 전체 규모별·부처별 예산 현황을 직관적으로 파악할 수 있는 단일 페이지를 구현합니다.

## 데이터

| Source | Format | Records |
| --- | --- | --- |
| `data/budget_data.csv` | → `dashboard/data.json` | 533건, 41부처 |

Python 스크립트로 CSV를 JSON으로 변환하여 `dashboard/` 폴더에 저장합니다.
